# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-24

### Added

- Initial release
- `add_geotag` tool - Add GPS and location metadata to photos
- `geocode_address` tool - Convert addresses to GPS coordinates
- `reverse_geocode` tool - Convert GPS coordinates to addresses
- `rename_for_seo` tool - SEO-optimized file renaming
- `batch_geotag` tool - Batch process directories
- `read_geotag` tool - Read existing location metadata
- OpenStreetMap Nominatim integration for free geocoding
- ExifTool integration for metadata operations
- EXIF, IPTC, and XMP metadata support
- Comprehensive README with SEO optimization
